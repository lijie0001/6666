# Telegram AI 机器人 - 完整流程手册

## 一、项目初始化

### 1. 创建 Python 项目

```powershell
cd d:\python
```

### 2. 创建虚拟环境

```powershell
python -m venv venv
```

### 3. 激活虚拟环境

**PowerShell：**
```powershell
.\venv\Scripts\Activate.ps1
```

**CMD：**
```cmd
venv\Scripts\activate.bat
```

激活成功后命令行前会出现 `(venv)`。

### 4. 安装依赖

```powershell
pip install -r requirements.txt
```

国内网络可加镜像：
```powershell
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 5. 退出虚拟环境

```powershell
deactivate
```

---

## 二、Telegram 机器人配置

### 1. 获取 Bot Token

1. 在 Telegram 搜索 **@BotFather**
2. 发送 `/newbot`
3. 按提示输入机器人名称和用户名（以 `bot` 结尾）
4. 复制获得的 Token

### 2. 配置 .env 文件

在项目根目录创建 `.env` 文件：

```
TELEGRAM_BOT_TOKEN=你的Token
TELEGRAM_PROXY=http://127.0.0.1:7890
OPENAI_API_KEY=你的OpenAI_Key
```

- `TELEGRAM_PROXY`：国内需代理，Clash 默认 7890
- `OPENAI_API_KEY`：用于 AI 回复，可选

### 3. 本地运行机器人

```powershell
cd d:\python
.\venv\Scripts\Activate.ps1
python telegram_bot.py
```

## 三、推送到 GitHub

### 1. 安装 Git

https://git-scm.com/download/win

### 2. 配置 Git（首次）

```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

### 3. 在 GitHub 创建仓库

1. 打开 https://github.com → 登录
2. 右上角 **+** → **New repository**
3. 填写仓库名，选择 Public，**不要**勾选 Add README
4. 点击 Create repository

### 4. 推送代码

```bash
cd d:\python

# 初始化
git init

# 添加文件
git add telegram_bot.py requirements.txt Procfile runtime.txt Dockerfile .dockerignore .gitignore DEPLOY.md

# 提交
git commit -m "Telegram AI bot"

# 添加远程（替换成你的）
git remote add origin https://github.com/用户名/仓库名.git

# 查看远程
git remote -v

# 推送（master 推送到 main）
git branch -M main
git push -u origin main
```

### 5. 若 remote 已存在

```bash
git remote remove origin
git remote add origin https://github.com/用户名/仓库名.git
```

### 6. 若分支是 master，GitHub 默认 main

```bash
git push -u origin master:main
```

### 7. 推送失败时（网络/SSL 问题）

**配置代理：**
```bash
git config --global http.proxy http://127.0.0.1:7890
git config --global https.proxy http://127.0.0.1:7890
```

**或改用 SSH：**
```bash
ssh-keygen -t ed25519 -C "邮箱" -f ~/.ssh/id_ed25519 -N ""
cat ~/.ssh/id_ed25519.pub
# 复制到 GitHub → Settings → SSH and GPG keys

git remote set-url origin git@github.com:用户名/仓库名.git
git push -u origin master:main
```

## 四、部署到 Railway

### 1. 连接 GitHub

1. 打开 https://railway.app，用 GitHub 登录
2. **New Project** → **Deploy from GitHub repo**
3. 选择你的仓库

### 2. 配置环境变量

进入服务 → **Variables** → **+ New Variable**：

| 变量名 | 值 |
|--------|-----|
| TELEGRAM_BOT_TOKEN | 你的 Bot Token |
| OPENAI_API_KEY | 你的 OpenAI Key |

### 3. 配置检查

- **Pre-deploy command**：留空（不要填 `npm run xxx`）
- **Start command**：`python telegram_bot.py`
- **重启策略**：On Failure
- **Serverless**：关闭
- **Builder**：Dockerfile

### 4. 部署

推送代码后 Railway 自动部署，或手动点击 **Redeploy**。

## 五、Telegram 群组操作

### 1. 创建群组

**手机：**
1. Telegram → 右上角 ✏️ 新建
2. 新建群组
3. 选择成员 → 输入群名 → 创建

**电脑：**
1. 菜单 → 新建群组
2. 选择成员 → 输入群名 → 创建

### 2. 添加机器人到群组

1. 打开群组 → 点击群名称
2. 添加成员
3. 搜索机器人用户名（如 `@zheshiwhat_bot`）
4. 添加

### 3. 群组隐私模式（搜不到机器人时）

1. 找 @BotFather
2. 发送 `/mybots` → 选择你的机器人
3. Bot Settings → Group Privacy → **Turn off**

## 八、群组爬虫（telegram_crawler.py）

### 1. 获取 API 凭证

1. 打开 https://my.telegram.org
2. 用手机号登录
3. API development tools → 创建应用 → 获得 API_ID、API_HASH

### 2. 配置 .env

```
TELEGRAM_API_ID=你的API_ID
TELEGRAM_API_HASH=你的API_HASH
TELEGRAM_PHONE=+8613800138000
CRAWLER_SOURCE=-1001234567890  # 源群组 ID
CRAWLER_TARGET=-1009876543210  # 目标群组 ID
```

### 3. 获取群组 ID

```powershell
python get_group_id.py
```

运行后会列出你加入的群组及 ID，复制到 .env。

### 4. 运行爬虫

```powershell
python telegram_crawler.py
```

首次运行会要求输入 Telegram 验证码。爬虫会监控源群组新消息并转发到目标群组。

**注意**：需先用自己的账号加入源群组和目标群组。
